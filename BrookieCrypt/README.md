**BrookieCrypt es un programa que sirve para comprimir, cifrar y crear sumas de comprobación
de un archivo o una carpeta, todo esto en conjunto.**

**Algoritmo(s) de compresión utilizado(s):**
```
zip
```

**Método de cifrado:**
```
Simétrico
```

**Dependencias:**
```
zip
GnuPG (GPG)
GNU Core Utils
```
